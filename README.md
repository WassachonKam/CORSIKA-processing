# CORSIKA Processing Pipeline

This repository contains scripts and notebooks for organizing, decoding, and plotting simulation data.  
All simulation folders share the same subdirectory structure: `.../particle/energy/angle/`



## Folder Description

All folders follow the same subdirectory structure, e.g.  

1. **/DATfiles**  
   Stores `DATxxxxxx` files generated by `gather_DAT.sh`.

2. **/Particles**  
   Stores particle number files generated by `corsikaReader.cpp`.

3. **/rawdata**  
   Stores raw simulation data.

## File Preparation

1. Set path (in `./gather_DAT.sh`, and  `functions.py`) to the simulation raw data.
2. Set input parameters in `functions.py` 
3. Gather `DATxxxxxx` files from each subdirectory in raw data path into `/DATfiles` using:
   ```bash
   ./gather_DAT.sh
4. Decode Binary DATxxxxxx Files `make` and `./corsikaReader DATfiles/.../DATxxxxxx --thinned`. This step generates decoded particle files in the `/Particles` directory.
